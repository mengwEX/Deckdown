# @deckdown/schema

Shared TypeScript contracts and validation helpers for Deckdown `.dd` decks.

## Install

```bash
npm install @deckdown/schema
```

## Usage

```ts
import { DeckDocument, hasErrors } from "@deckdown/schema";
```

This package is part of [Deckdown](https://github.com/mengwEX/Deckdown) and is released under the MIT License.
